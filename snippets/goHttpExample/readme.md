# Go HTTP Example

This go example uses mux of the gorilla webkit.

## Set Up

 - execute `go mod download`

## Run

 - verify that no other process runs on port 8080
 - execute `go run .`

## Try

 - http://127.0.0.1:8080/hello?name=Studi%20Guide
 - http://127.0.0.1:8080/hello/Studi%20Guide
 - http://127.0.0.1:8080/static/style.css
